# Tableau périodique - Bac Pro

Projet pédagogique simple pour visualiser les éléments chimiques.

## Objectifs
- Lire facilement la masse atomique
- Comprendre l'organisation du tableau périodique
- Interaction simple (clic)

## Lancer le projet
Ouvrir index.html dans un navigateur.

## Améliorations possibles
- Ajouter tous les éléments
- Colorer les familles (métaux, gaz nobles…)
- Ajouter une recherche
